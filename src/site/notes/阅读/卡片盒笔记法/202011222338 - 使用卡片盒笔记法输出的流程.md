# 内容
1. 整体看一下相关的知识
2. 为写作列出需要的大纲
3. 把永久笔记放到大纲中
4. 使用更多的知识（研究）填补空白，重复3-4步，直到满意为止
5. 根据提纲复制笔记到文档中
6. 重写你的手稿
