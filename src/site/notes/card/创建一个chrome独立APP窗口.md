# 创建一个chrome独立APP窗口

创建出的窗口本质还是一个chrome网页，但是比较方便使用

按步骤操作：
1. 点击chrome右上角的三个点
2. 点击保存并分享
3. 点击创建快捷方式
4. 在弹窗中勾选`在窗口打开`，点击创建
5. 即可得到一个独立APP窗口

![../attachs/clop_2024-02-27_159.png](/img/user/attachs/clop_2024-02-27_159.png)

弹窗出来之后，选择 `在窗口中打开`
![../attachs/clop_2024-02-27_160.png](/img/user/attachs/clop_2024-02-27_160.png)


就会得到一个独立软件一样的百度
