# 接入外置键盘时如何关掉内置键盘

- 下载： [Karabiner-Elements](https://karabiner-elements.pqrs.org)
- 连接外接设备后，点击 `Disable the built-in keyboard while this device is connected` 后面的按钮，即可达到此功能

- 我主要是用在外置键盘放到mac键盘上面，防止误触
