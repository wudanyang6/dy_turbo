# 202011201301 - 基础概念

## 四种卡片

1. ==临时卡片==，把自己的==想法==记录下来
2. ==文献卡片==，看书或者其他信息，觉得自己能用到就记录下来，注意，用自己的话来描述，不能复制粘贴或者剪藏
3. ==永久卡片==，将临时笔记和文献笔记组织成文，找出笔记之间的联系，将笔记放到卡片盒中
4. ==项目卡片==

1. 知识卡片盒
