# 202011230004 - 如何写出容易理解的卡片

1. 一个好的标题，可以是一种注释
2. 第一句话应该是关于此卡片的
3. 写的话要简单
4. 你是在教育你的卡片盒，他会在以后的某个时间点出来跟你对话
5. 想象一下，你是在跟以后的自己在对话
